The purpose of DataSafe
DataSafe is a tool for creating, sharing and browsing data cards for datasets used for AI or in AI systems. It aims to provide dataset creators with a method of reflection as they build datasets through a tool which should be used alongside dataset creation. As well as this, data cards are to be employed by those using such datasets in order to make ethical decisions on what datasets to use for projects. Data cards can be used to justify decisions on which dataset is best suited for a given application and can be provided as a means of transparency in data driven projects.
Background
Creating a data card
The process of creating a data card should be easy and interactive. The creation guide was designed to be incorporated into the process of creating a dataset. Because of this, the questions aim to follow the timeline of creating a dataset so both can be worked on in parallel. This also provides a method of reflection for any dataset creators during this process.

Data cards were an innovation in promoting ethical AI inspired by model cards, a similar tool but for use with AI models. This idea of model cards was promoted by Margaret Mitchell and co in 2018 to make fairer and more ethical use of AI systems easier than it had been previously. Data cards use this idea and deploy it at an earlier stage in the development of such AI systems.
